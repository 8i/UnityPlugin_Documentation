--------------

    2. Requires Unity 5.4 or greater

Using LightProbes to Light HvrActors
------------------------------------

    This example shows how to sample Unity LightProbes and affect HvrActors to give the apperance of realtime lighting.

    2.  Requires Unity 5.4 or greater
